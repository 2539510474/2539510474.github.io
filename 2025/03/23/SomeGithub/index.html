WechatRealFriends

这是什么？

你是否想知道你的微信里有多少人已经删掉你了？或者拉黑你了？

功能

快速检测你的微信有没有单向好友（删除你的和拉黑你的）,并自动将其添加到标签中(之后可在电脑端一键清除这些人),理论上可以支持检测上万的好友

PS:本软件不会打扰到好友,好友不会收到任何提示,使用非常简便

使用指南

在releases中下载最新构建版本 点击跳转到releases页面

解压后看txt文档里的说明操作

gpt_academic

这是什么？

还在苦恼读EN论文下了个PDF找百度翻译找DeepL又要钱效果又不好吗？

功能

Link发现本项目是因为有读文献的需求，而我的英文水平停留在初二（= =）。于是乎百度翻译，DeepL我都尝试过，效果并不好，普通的大模型又无法翻译那么长的文献，最后瞧上了该项目的arxiv论文翻译功能

arxiv论文翻译：非常方便，部署之后用Gui界面填写arxiv论文网址即可开始翻译

该项目的功能非常之多，Link并未详细探索，详见项目地址

该项目前置需求：TeX环境，买上一个大模型几块钱的API，python环境（这个会没有吗

使用指南

直接跟着项目readme做就行

先找一个你喜欢的文件夹clone一下

‘’’shell
git clone –depth=1 https://github.com/binary-husky/gpt_academic.git
cd gpt_academic
‘’’

下载之后找到config.py，配置API KEY

比如我用的deepseek的API，购买完之后直接复制到config.py里的APIKEY里，然后LLM_MODEL处选择”deepseek-chat”, “deepseek-coder”, “deepseek-reasoner”,里的一个，由于deepseek不需要翻墙所以代理处，USE_PROXY不需要填写

之后运行main.py

‘’’shell
python main.py
‘’’

可能会报错“Latex没有安装”，询问大模型如何安装即可
